以下只针对个人不熟悉的知识点进行举例

基础部分

中文乱码问题

在head里添加以下代码

<meta http-equiv="Content-Type" content="text/html; charset=utf-8">

加粗

<b></b> （逐渐被弃用）
<strong></strong>

预格式

<pre></pre>
在网页上显示代码

删除线效果

<del></del>
delete
<s></s>
s标签不建议使用，因为很多浏览器不支持s标签

下划线

<ins></ins>
<u></u>
不建议使用u标签

分割线

<hr>

居中属性

<h1 align="center">居中标题</h1>

替换图片上的文字

如果图片不存在，默认会显示一个缺失图片，这是不友好的
所以可以加上alt属性。
当图片存在的时候，alt是不会显示的
当图片不存在的时候，alt就会出现

<img src="#" alt="这个是一个图片" />

在新的页面打开超链

target属性，target意思是目标，目的

<a href="http://www.12306.com" target="_blank">http://www.12306.com</a>

超链上的提示文字

title当鼠标放在超链上的时候，就会弹出提示文字

<a href="http://www.12306.com" title="跳转到http://www.12306.com">www.12306.com</a>

使用图片作为超链

<a href="http://www.12306.com">
<img src="example.gif"/>
</a>

表格

行列

<table>标签用于显示一个表格
<tr> 表示行
<td> 表示列又叫单元格,列
<th> 表格里面的标题，位置上相当于<td>
3行2列表格：

<table>
  <tr>
      <td>1</td>
      <td>2</td>
  </tr>

  <tr>
      <td>3</td>
      <td>4</td>
  </tr>

  <tr>
      <td>a</td>
      <td>b</td>
  </tr>

</table>

table边框

设置table的属性border的宽度为1，显示表格
<table border="1" width="200px">
width设置整个表格的宽度

单元格td宽度绝对值(重点)

设置第一行的td属性width，那么这个td下面的所有td自动继承该宽度
其他的单元格的宽度由table总宽度和第一个单元格td的宽度决定

<table border="1" width="200px">
  <tr>
      <td width="50px">1</td>
      <td>2</td>
  </tr>

  <tr>
      <td>3</td>
      <td>4</td>
  </tr>

  <tr>
      <td>a</td>
      <td>b</td>
  </tr>

</table>

单元格宽度相对值

设置td的属性width为百分数

<table border="1" width="200px">
  <tr>
      <td width="80%">1</td>
      <td>2</td>
  </tr>

  <tr>
      <td>3</td>
      <td>4</td>
  </tr>

  <tr>
      <td>a</td>
      <td>b</td>
  </tr>

</table>

td也可以设置align属性

<td align="center">3</td>
<td align="right">a</td>

单元格垂直对齐

设置td的属性valign

<table border="1" width="200px">
  <tr>
      <td width="50%" valign="top" >1</td>
      <td>
             2   <br/>
             2   <br/>
             2   <br/>
      </td>
  </tr>

  <tr>
      <td valign="middle"  >3</td>
      <td>
             4   <br/>
             4   <br/>
             4   <br/>
      </td>
  </tr>

  <tr>
      <td valign="bottom" >a</td>
      <td>
             b   <br/>
             b   <br/>
             b   <br/>
      </td>
  </tr>

</table>

横跨两列, 水平合并

设置td的属性colspan,col行

<table border="1" width="200px">
  <tr>
      <td colspan="2" >1，2</td>
  </tr>
  <tr>
      <td>3</td>
      <td>4</td>
  </tr>
  <tr>
      <td>a</td>
      <td>b</td>
  </tr>
</table>

横跨两行, 垂直合并

设置td的属性rowspan,row列

<table border="1" width="200px">
  <tr>
      <td rowspan="2">1,3</td>
      <td>2</td>
  </tr>

  <tr>
      <td>4</td>
  </tr>

  <tr>
      <td>a</td>
      <td>b</td>
  </tr>

</table>

表格的背景色

设置tr或者td的属性bgcolor

<table border="1" width="200px">
  <tr bgcolor="gray">
      <td width="50%">1</td>
      <td>2</td>
  </tr>

  <tr>
      <td>3</td>
      <td>4</td>
  </tr>

  <tr>
      <td>a</td>
      <td  bgcolor="pink">b</td>
  </tr>

</table>

列表

有序列表

可以理解为orderlist，有序的列表
<ol></ol>

无序列表

可以理解为unorderlist，无序的列表
<ul></ul>

补充

小细节：div和span的区别

div是块元素，即自动换行
常见的块元素还有h1，table，p
span是内联元素，即不会换行
常见的内联元素还有img，a，b，strong

字体元素

font常用的属性有 color（不是font-color）和size, 分别表示颜色和大小

<font color="blue" size="+2">蓝色大2号字体</font>

图像提交按钮

这里的图像按钮具有提交表单的功能，相当于submit

<input type="image" src="example.gif">

在HTML中引用css样式

这个老是忘

<link rel="stylesheet" href="css的路径" type="text/css">
