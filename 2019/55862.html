Ubantu系统安装

关于/swap分区
连接：https://www.jianshu.com/p/12e25ea172de

安装成功后的样子

禁用网卡驱动

列出所有安装网卡

sudo lshw -C network

找到要禁用的网卡，在其中找到driver后的属性，复制

sudo gedit /etc/modprobe.d/blacklist.conf

在最后一行写入（你想要禁用的设备的驱动）

# ----------------------------
# Disable WIFI driver
blacklist rtl8723be

安装显卡驱动

ubantu默认是使用Intel显卡，也就是核显，体验不是那么舒服，想开启独立显卡，怎么做呢？
开启独显方法很多，我这里介绍个对新手很友好的方式：
首先 添加Nvidia官方源

sudo add-apt-repository ppa:graphics-drivers/ppa


提示输入密码，然后输入的密码看不到（其实密码已经输入上去了），直接回车就行了！
PS：Unix思想，没有消息就是好消息嘛！

然后更新源

sudo apt-get update

然后左下角启动器菜单打开软件和更新，点击附加驱动，这时候系统会加载NVIDIA官方的显卡驱动和uabntu系统自带的nouveau显卡驱动
一般选择合适的驱动版本号，点击应用更改即可

（如果不添加以上NVIDIA源，这里也会显示很多驱动，但都是老旧驱动，跟不上新内核，所以安装NVIDIA的驱动比较好）

然后。。。漫长等待
（这个时候，可以去下载一些开发软件什么，当然，安装不上，因为显卡驱动正在安装）

安装成功后重启，去设置->详细信息->about，查看 图形 是不是你的独显

安装搜狗输入法

首先 去搜狗输入法官网下载Linux版本的输入法
然后 添加源

sudo add-apt-repository ppa:fcitx-team/nightly

然后 更新源

sudo apt-get update

接着 直接安装搜狗输入法.deb文件（没有添加源是安装不上的）

安装成功 再去左下角启动器菜单里找到语言支持 打开
一般会提示 语言支持没有安装完整 这个安装一下也好，也可以等会安装，点击稍后提醒
然后找到键盘输入法系统，选择fcitx， 然后 重启！（一定要重启，不然不会生效）

大功告成！
其实ubantu自带的有IBus中文输入法，但是体验太差

这个时候右上角依然有烦人的IBus，其实这时候搜狗已经接管了系统输入，强迫症会看着很不舒服（一定不要直接把IBus卸载，不然会有很多麻烦）
解决办法，在状态栏右键搜狗退出，然后在启动器里打开Fcitx，问题解决！
如果还不行再重启下就好了

安装jdk

安装jdk有两种方式：
apt安装和下载安装
这里介绍下载安装和配置环境变量

  1. 下载
    首先jdk常用的有两种open jdk和Oracle jdk
    我们安装Oracle jdk
    （如果不知道怎么下载，点我下载jdk8u202 ）
    选择Linux 64位的.tar.gz文件
    在jdk所在文件夹内，右键在终端打开，输入以下命令：
  2. 创建目录：
sudo mkdir /usr/lib/jvm
  1. 解压到该目录：
sudo tar -zxvf jdk-8u201-linux-x64.tar.gz -C /usr/lib/jvm
  1. 修改环境变量：
sudo gedit /etc/profile

执行命令之后会弹出文本编辑器，在文本的最后一行加上以下内容：

#--------------------------------------------
set java environment
JAVA_HOME=/usr/lib/jvm/jdk1.8.0_201
JRE_HOME=$JAVA_HOME/jre     
CLASS_PATH=.:$JAVA_HOME/lib/dt.jar:$JAVA_HOME/lib/tools.jar:$JRE_HOME/lib
PATH=$PATH:$JAVA_HOME/bin:$JRE_HOME/bin
export JAVA_HOME JRE_HOME CLASS_PATH PATH

注意：其中 JAVA_HOME 请根据自己的实际安装路径及 JDK 版本配置填写！

让修改生效，重启电脑
或者使用以下命令（立即生效）：

source /etc/profile
  1. 测试
    看jdk版本是否正确
java -version

或者再试试java javac

卸载软件

sudo apt-get remove --purge 软件名称     
sudo apt-get autoremove --purge 软件名称 
sudo apt-get -y autoclean 软件名称 
sudo apt-get -y clean 软件名称

//remove卸载软件，–purge清除相关配置文件，autoremove清理不需要的依赖包
autoclean清理安装软件时留下的缓存程序软件包（只清理过时（已经被卸载的软件）的包）
clean清理所有下载的安装包
清理软件残留命令

dpkg -l |grep ^rc|awk '{print $2}' |sudo xargs dpkg -P

检查你的电脑有哪些显卡

lspci -k | grep -A 2 -i "VGA"
