持续更新中。。。

软件系列

markdown编写软件

  1. Typora

    • 支持Windows、Mac、Linux
    • 暂时免费


    Typora以后会用在商业（收费），但目前是免费beta公测的。是所见即所得类型的编辑器，即敲出代码-回车-立即显示样式，而且还是跨平台的，Win、Mac、Linux都支持。

  2. Marktext

    • 支持Windows、Mac、Linux
    • 免费、开源

    也是所见即所得类型的编辑器，markdown编辑器我只钟爱这一种，其他的编辑器一般都是分两页（左边敲字，右边显示），太占屏幕，Marktext是跨平台的，这两款都很值得推介。

    值得说的是，Marktext是免费开源的，在GitHub上有项目，但好有好多小bug，不过现在应该修复得差不多了。

  3. 马克飞象

    • 网页版（支持各种平台）
    • 免费

    马克飞象是一款网页版的Markdown编辑器，所以在任何平台都可以用，只要有浏览器就可以，并且还可以配合 印象笔记（Evernote）强大的存储和同步功能。也有离线版的APP。

轻量级文本编辑器

  1. Sublime Text 3

    • 支持Windows、Mac、Linux
    • 收费，但可以无限试用（有弹窗提示）

  1. VScode

    • 支持Windows、Mac、Linux
    • 免费

    采用electron开发（基于HTML的一种技术），所以性能较弱（启动慢），有丰富的插件，社区较为活跃

远程控制软件

  1. Teamviewer

    • 支持Windows、Mac、Linux、Android、IOS
    • 个人版免费

    个人比较推崇的一款软件，这个软件的远程连接的稳定性、流畅度都非常好，比一些QQ之类的远程好太多了。。

  1. Remote Desktop

    • 仅支持Windows，（不过在其他平台上有控制端）
    • 免费

    Windows自带的远程连接软件，Windows10家庭版是没有此功能的，只有专业版及以上才有。

    该软件有一点，值得注意，远程连接时，是使用Windows账户登陆的，也就是说，被远程控制端的账户会被注销。

    另外这款软件的流畅度是非常棒的，有微软的优化

  2. NoMachine

    • 支持Windows、Mac、Linux、IOS、Android、树莓派、ARM

    • 免费

      NoMachine有自己的一种 NX协议 技术 ，采用了新的远程协议，把x桌面的数据经过提取，仅传送一些参数，而且通过ssh来进行数据加密传送，因此相比VNC或直接XDMP具有很快的速度和较高的安全性能。

    性能甚至可以远程打游戏。

  3. VNC

    VNC (Virtual Network Console)是虚拟网络控制台的缩写。它 是一款优秀的远程控制工具软件，由著名的 AT&T 的欧洲研究实验室开发的。VNC 是在基于 UNIX 和 Linux 操作系统的免费的开源软件，远程控制能力强大，高效实用，其性能可以和 Windows 和 MAC 中的任何远程控制软件媲美。 —百度百科

    没使用过，不做评论

  4. 蒲公英

    • 支持Windows、Mac、Linux、Android、IOS
    • 免费试用版最高限速300Kbps

    国产软件，没使用过，不做评论

图床上传软件

  1. PicGo

    • 支持Windows、Mac、Linux
    • 免费开源

    支持微博图床（微博图床从 2019 年 4 月开始进行防盗链，不建议继续使用）、七牛图床、腾讯云COS、又拍云、GitHub、SM.MS、阿里云OSS、Imgur等各大图床

    使用上也很方便，上传后会自动在粘贴板生成Markdown等格式的链接
